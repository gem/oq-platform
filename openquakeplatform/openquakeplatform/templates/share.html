The OpenQuake Platform includes a number of tools allowing users to contribute to the existing datasets.

Online Data Capture and data Collection
  • The Faulted Earth capture tool allows users to share information about Active Faults. Please read the guidelines before using the tool.
  • The Physical Vulnerability suite allows users to explore and contribute to a comprehensive global vulnerability database that includes fragility, vulnerability, damage-to-loss functions and capacity curves for different types of structure.
  • The Building Classification Survey tool allows users to share information regarding the building stock of a given region.
Offline Inventory Capture Tools
The following tools were developed as part of the Inventory Data Capture Tools global component:
  • A Building Data Capture application for Android phone or tablet is available for installation via Google Play.
  • Spatial Inventory Data Development (SIDD) Tool
GEM Building Taxonomy Tools
The GEM Building Taxonomy is used in a number of GEM databases to classify building types.
  • TaxTweb Graphical tool for editing GEM Taxonomy strings, web version
  • TaxT Graphical tool for editing GEM Taxonomy strings, Windows only
Upload:
The OpenQuake platform also allows users to upload arbitrary documents and GIS map layers:
  • Documents
  • Layers
